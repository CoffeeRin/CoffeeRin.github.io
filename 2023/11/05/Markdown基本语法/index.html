markdown:通用html标签

标签后要加空格

块级元素：至少独占一行
行内元素：只占据自身所需空间

# 标题1
## 标题2
### 标题3

效果：

标题1

标题2

标题3

- [ ] b
- [x] c

  3. c

  • A
